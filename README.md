# 스네이크 게임 (HTML 한 파일)

> CSS와 JS를 모두 포함한 **단일 HTML 파일**로 동작하는 스네이크 게임입니다.  
> 방향키(↑ ↓ ← →)로 조작하고, **Space**로 일시정지/재개, **R**로 재시작할 수 있어요.

---

## ✨ 특징(Features)

- **한 파일(HTML)** 안에 **CSS + JS** 모두 포함  
- **방향키** 조작, **Space**(일시정지/재개), **R**(재시작)  
- **속도 조절 슬라이더**(5~20 fps) — 실시간 적용  
- **역방향 전환 방지**, **자기 몸/벽 충돌 감지**  
- **고해상도 화면(DPR) 스케일링** 및 반응형 캔버스  
- **오버레이 안내(시작/일시정지/게임오버)**, HUD 점수 표시  
- 옵션으로 **벽 통과(워프)** 모드 전환 가능

---

## 🚀 실행 방법(Quick Start)

1. 저장소를 클론하거나 ZIP 다운로드  
2. `snake.html`을 더블 클릭하여 **브라우저로 열기**  

> 별도 서버나 빌드 과정이 필요 없는 **정적 단일 파일**입니다.

---

## 🎮 조작법(Controls)

- **이동**: ↑ ↓ ← →  
- **일시정지/재개**: `Space`  
- **재시작**: `R`  
- **속도 조절**: 화면 상단 슬라이더(5~20 fps)
